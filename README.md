# 🚀 AutoFlow AI: Smart Workflow Automation with n8n  

## 📌 Overview  
AutoFlow AI is a workflow automation project built using n8n that connects Google Forms, Google Sheets, and Gmail to create a seamless, automated system. It eliminates manual work by automating data collection, processing, and communication in real time.

This project demonstrates how low-code tools can be used to design scalable and efficient business workflows.

---

## ⚙️ Features  
- Automated Data Collection using Google Forms  
- Real-Time Data Storage in Google Sheets  
- Workflow Automation powered by n8n  
- Automated Email Responses via Gmail  
- Custom Logic & Conditions for dynamic workflows  
- Instant Triggers & Actions  

---

## 🏗️ Tech Stack  
- n8n – Workflow automation tool  
- Google Forms – Input collection  
- Google Sheets – Data storage & tracking  
- Gmail API – Automated email communication  

---

## 🔄 Workflow Architecture  
1. User submits a response through Google Forms  
2. Data is automatically stored in Google Sheets  
3. n8n monitors new entries in the sheet  
4. Workflow triggers based on predefined logic  
5. Personalized email is sent via Gmail  

---

## 🛠️ Setup Instructions  

### 1. Clone the Repository  
git clone https://github.com/your-username/AutoFlow-AI.git  
cd AutoFlow-AI  

### 2. Set Up n8n  
- Install n8n locally or use cloud version  
- Import the provided workflow JSON file  

### 3. Connect Integrations  
- Connect your Google Account  
- Enable APIs for Google Sheets, Gmail, and Google Forms  

### 4. Configure Workflow  
- Update Sheet IDs, Form IDs, and Email templates  
- Set trigger conditions as required  

### 5. Run the Workflow  
- Activate the workflow in n8n  
- Test by submitting a form response
